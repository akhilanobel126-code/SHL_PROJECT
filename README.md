# SHL Assessment Recommender Agent

## Overview
This project builds a conversational AI agent that recommends SHL assessments using semantic search.

## Features
- Web scraper for SHL catalog
- FAISS vector search for semantic retrieval
- FastAPI chat interface
- JSON structured recommendations

## API Endpoints
- GET /health → system status
- POST /chat → recommendation engine

## Tech Stack
- Python
- FastAPI
- FAISS
- Sentence Transformers
